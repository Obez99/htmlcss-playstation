# Playstation

**Description**:<br>
In this project I made the clone of the Playstation website using bootstrap.<br>

**Features**:<br>
- Bootstrap grid layoud;
- Hoverable buttons and elements;
- Clickable and animated dropdown menu in navbar;
- Responsiveness;
